# Space Cat Food Catcher 13

Статическая HTML5 Canvas аркада: милые космические животные уворачиваются от бомб, собирают еду/бонусы и открываются по уровням 1–10.

## Запуск

Откройте `index.html` в современном браузере. Для полной загрузки JSON-карт ассетов можно также открыть папку через любой локальный static server, но игра содержит fallback-таблицы и playable при прямом открытии файла.

## Управление

- WASD / стрелки — движение.
- Space — рывок.
- P / Esc — пауза.
- На телефоне: перетаскивание по экрану, кнопка ⚡ для рывка.

## Ассеты и звук

Игра пытается загрузить локальные Grok raster assets из `data/raster-asset-map.json` и размеры из `data/object-size-map.json`; если браузер блокирует `fetch` при `file://`, используются локальные пути fallback. Звук использует только локальные Kenney `.ogg`, если они доступны; procedural audio не используется.

## Прогресс

Уровень, XP, лучший счёт, выбранное животное и настройки сохраняются в `localStorage`.
